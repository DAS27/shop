## Description
Test e-commerce shop

## Requirements
- PHP 7.4
- Extensions: sqlite3
- SQLite for local
- Composer
- Postman

## Setup
````
$ make setup
````

## Launch local
Server will be available at http://localhost:8000/
````
$ make start
````
